Templates can be configured using config files.

Each template can have an optional config file. The file is always named #Tmplt.<name>.cfg.

Example

[Template]
Parent=Default

[Template]

Parent

Specifies the name of a parent template. If a parent template is used, cascading templates are enabled and the templates are merged from this template to the parent template. If the parent template also has a parent, the cascade will occur at another level. See templates for more details. If no Parent value exists, no parent template will be used. A value . specifies that a parent template of the same name should be used.
