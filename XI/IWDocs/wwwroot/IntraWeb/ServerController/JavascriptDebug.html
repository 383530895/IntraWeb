Controls debug output and other options in the IntraWeb javascript libraries. Usefulf or AJAX and HTTP debugging.

True

Enables debug output.

False (Default)

Disables debug output.

Destination

The output is directed at the first available method in the following order:

  1. On FireFox with FireBug, the Firebug console will be used.
  2. A TIWText or TIWMemo named INTRAWEBLOG.
  3. A popup window.
