property AppName: string;

AppName is a short string used to identify the application. AppName is used to:

  • Identify the application when installed as a service and should not conflict with other services installed.
  • Create a unique cookie name for tracking the user session
  • Display debugging information during unhandled exception.

AppName must not contain spaces or any special characters. AppName must be a valid identifier and should only contain letters and numbers. AppName must not start with a number.
