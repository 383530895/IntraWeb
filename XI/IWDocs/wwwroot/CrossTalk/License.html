Per Developer

CrossTalk is licensed per developer, not per machine. This means that if you purchase a CrossTalk license "1 Developer", CrossTalk may be used by a single developer. That single developer may install CrossTalk on multiple machines, but CrossTalk can only be used by that single developer.

Redistributable Files

You may redistribute:

  • CrossTalkMiddle.dll
  • Binary files (exe, dll, packages, etc) which include source code generated by CrossTalk.
  • VC Redistributable Files - These files are provided by Microsoft.

Restricted Files

You may not redistribute any of the following:

  • Any source file genereated by CrossTalk.
  • The CrossTalk Class Generator utility.
  • Your CrossTalk license key.
