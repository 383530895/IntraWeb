This documentation project is hosted as one of the subprojects of the IntraWeb XI Demos project at CodePlex. Here are the steps to contribute to these docs.

  1. Request access to the IntraWeb XI Demos project.
  2. The documentation project exists in the Atozed\IWDocs directory. The easiest way is to access it via SVN at https://IWXIDemos.svn.codeplex.com/svn. If you are not familiar with SVN, we suggest TortoiseSVN.
  3. Modify the content files as needed in the wwwroot subdirectory.
  4. Check in your changes.
  5. Notify us of the checkin, and we will publish it using the assigned Server Manager.
